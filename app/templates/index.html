Iris Dataset Visualization

About this Visualization

This scatterplot shows the relationship between sepal length and sepal width for three species of iris flowers: Setosa, Versicolor, and Virginica. Each point represents a single flower, with the color indicating the species.

The visualization was created using:

  • Polars: A fast DataFrame library for data manipulation
  • Plotnine: A Python implementation of the grammar of graphics (similar to R's ggplot2)
  • Flask: A lightweight web framework for Python
